## Подготовка
- Требуется Python 3.12 для работы приложения.
- Необходимо внести изменения в файл, где указать VK API. Постоянный ключ, не чувствительный к ip адресу, можно получить, создав приложение ВК в разделе для разработчиков (подробнее [Здесь](https://dev.vk.com/ru/api/access-token/getting-started)). id группы ВК уже указан
## Запуск
Для первого запуска необходимо запустить **setup.bat** (создаст окружение и установит зависимости).

Для последующих запусков используйте **start.bat** или **run.bat** (быстрый запуск без установки зависимостей).

Скрипт парсера запускается кнопкой **Обновить таблицу**.
После каждого обновления, группа актуальных постов сортируется хронологически.

_При первом запуске может появится ошибка об отсутствии базы данных._

## Репозиторий (GitHub)
После клонирования создайте файл `.env` на основе `.env.example` и укажите в нём свой VK API ключ и ID группы.
